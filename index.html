Projects

Here are some of my most recent projects.

Ekho

A simple sheet music editor in its early stage. Built with React.js

  • Minimized the amount of HTTP requests made by using custom designed inline SVGs
  • Created custom CSS animations using strictly opacity and transforms in @keyframes
  • Simplifies User Experience by adjusting the staff & measures based on placed notes

Bust-A-Move

A clone of the video game with the same name. Built using Processing.js

  • Optimized the efficiency of the game by handling most of the game logic in a particular frame
  • Identified adjacent bubbles of the same color by a recursive breadth-first graph search
  • Built a grid system with logic to determine exactly where a bubble should be placed

Taskable

A full-stack website to organize your to-do lists. Built with Ruby on Rails

  • Implemented a dynamic search feature that fetches partial title matches on every keystroke using AJAX
  • Utilizes the React Router to create and maintain a single-page app while still directing users by updating the URL according to user navigation
  • Minimized the amount of user clicks by automatically saving a task object after every edit
